# whatismyip

A simple command line tool for looking up your public IP address. Utilises the extremely handy 
[ifconfig.co](https://ifconfig.co) from [mpolden](https://github.com/mpolden)


## Building

For mac/linux 
```
go build whatismyip.go
```

For windows
```

```

## Usage

on mac linux

```
$ whatismyip
9.27.54.12
```
